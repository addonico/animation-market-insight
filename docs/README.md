# Plain static website

This folder is the GitHub Pages version of the project. It uses only HTML, CSS, JavaScript and CSV.

## Publish with GitHub Pages

1. Push the repository to GitHub.
2. Open **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose the main branch and the `/docs` folder.
5. Save and wait for the public URL.

Before submission, replace the placeholder credit in `index.html` with your name and actual responsibilities.
